# QA Automation Challenges

Este repositório contém desafios técnicos de automação de testes,
abrangendo testes de API REST e testes Mobile.

## Estrutura do Projeto

- api-tests: testes automatizados de API
- mobile-tests: testes automatizados Mobile

## Tecnologias Utilizadas

- Robot Framework
- RequestsLibrary
- AppiumLibrary
- GitHub Actions
